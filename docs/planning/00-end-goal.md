# End Goal

Write one plain-language paragraph describing the finished product and quality bar after the first planning pass. Use it as the target for the roadmap and production-hardening passes.

## Current Draft

HostDeck is a local-first control plane for Ubuntu users who run many Codex CLI sessions: an Ubuntu host agent owns session lifecycle, output capture, input routing, status detection, pairing, permissions, and auditability, while phone-friendly clients give the user a mission-control view of which sessions need attention. The finished product should let one user safely monitor sessions, read useful recent output, send prompts and approved slash commands, use confirmed voice-to-text input, and fall back to raw terminal control when needed without becoming a generic SSH terminal; the quality bar is reliable real-work supervision, secure-by-default local operation, explicit remote-control trust boundaries, honest status signals, and a mobile UX centered on triage and safe steering.
