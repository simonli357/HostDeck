# End Goal

HostDeck is a local-first mobile control plane for one Ubuntu user running several Codex sessions. A supervised host service integrates with Codex through typed, version-checked contracts; a phone-first dashboard works while the phone and laptop are on different networks, shows which session needs attention, presents the actual conversation and approval state, and safely supports prompts plus the primary `/model`, `/goal`, and `/plan` controls. V1 reaches the loopback-only HostDeck service through a user-selected personal Tailscale profile and private Tailscale Serve HTTPS ingress, without a public laptop listener, router change, or manually installed CA. The laptop Codex TUI remains available for full control of the same thread, but the phone surface is not a terminal emulator. The finished product is secure by default, explicit about identity, active-tailnet and failure state, bounded in storage and resource use, recoverable across service and network-profile changes, and proven with real Codex, browser, phone-on-an-unrelated-network, packaging, and release evidence rather than package-local tests alone.
