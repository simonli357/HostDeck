# End Goal

HostDeck is a local-first mobile control plane for one Ubuntu user running several Codex sessions. A supervised host service integrates with Codex through typed, version-checked contracts; a phone-first dashboard shows which session needs attention, presents the actual conversation and approval state, and safely supports prompts plus the primary `/model`, `/goal`, and `/plan` controls. The laptop Codex TUI remains available for full control of the same thread, but the phone surface is not a terminal emulator. The finished product is secure by default, encrypted whenever it leaves loopback, explicit about trust and failure, bounded in storage and resource use, recoverable across service restarts, and proven with real Codex, browser, phone, packaging, and release evidence rather than package-local tests alone.
