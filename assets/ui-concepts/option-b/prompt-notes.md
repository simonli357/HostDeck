# Option B Prompt Notes

## Generation Mode

- Built-in `image_gen`.
- Use case: `ui-mockup`.
- Direction: Focus Rail.
- Project-bound final assets are stored beside this file.

## Shared Final Prompt Contract

- Render a production-realistic phone-first HostDeck operational app, not marketing, terminal, SSH, IDE, or desktop dashboard.
- Use deep neutral surfaces, balanced semantic colors, continuous state/event rails, stable system type, 8px spacing, 4-6px radii, and 44px targets.
- Make the structure distinct from Option A through grouped queues, semantic timelines, staged rails, and rail-backed sheets.
- Keep Mission Control and Session Detail as the only full-page routes; `/model`, `/goal`, and `/plan` remain primary structured controls.
- Separate browser-preload failure from HostDeck-renderable local/remote/access states.
- Never show LAN, certificates, tmux, raw fallback, remote profile switching, Serve mutation, remote unlock, identity, credentials, or secrets.
