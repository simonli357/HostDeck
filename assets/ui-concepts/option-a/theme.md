# Option A Theme

Name: Dense Operations Console

## Mood

- Compact, operator-focused, high-signal.
- Prioritizes fast scanning of many concurrent sessions and immediate safety state recognition.
- Best fit when session count and status triage matter more than spaciousness.

## Product Fit

- Strongly matches HostDeck as a power-user mission-control dashboard.
- Makes locked, LAN-disabled, unknown, stale, and failed states hard to miss.
- Keeps raw fallback visually secondary and explicitly gated.

## Must Avoid

- Becoming a generic terminal emulator.
- Letting dense rows shrink text below readable sizes on phone.
- Overusing dark charcoal so caution, success, and neutral states lose contrast.
